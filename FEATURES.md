# Features

## Calculator
- **Live Calculation** — Enter any two values (speed, pace, distance, or time) and all remaining fields calculate instantly, no submit button needed
- **Speed** — Input/output in mph and km/h
- **Pace** — Input/output in min/mile and min/km
- **Distance** — Input/output in miles and kilometres
- **Time** — Input/output in HH:MM:SS format
- **Smart Anchor System** — Tracks up to two user-entered groups; derives the third automatically
- **Centiseconds** — Time field supports centisecond precision

## Input
- **Per-field Clear Buttons** — Tap × on any field to clear just that value
- **Distance Shortcuts** — Quick-tap buttons for 800m, 1.5K, 3K, 5K, 10K, Half Marathon, and Marathon
- **Flexible Parsing** — Accepts bare numbers (e.g. "830" → 8:30), partial times, and standard formatted input
- **Haptic Feedback** — Subtle tap when fields auto-populate

## Saved Presets
- **Save Calculations** — Save any current set of values with a custom name
- **Named Presets** — Give each saved entry a label (e.g. "Marathon pace")
- **Restore Preset** — Tap a saved entry to reload its values into the calculator
- **Rename Preset** — Edit a preset's name after saving
- **Reorder Presets** — Drag to reorder the saved list
- **Delete Preset** — Swipe or use context menu to remove a saved entry
- **iCloud Sync** — Presets sync across all your iPhone devices via iCloud

## Sharing
- **Share / Copy Result** — Share a formatted plain-text summary (speed, pace, distance, time) via the system share sheet

## Widget
- **Home Screen Widget** — Small and medium sizes showing your most recent saved preset (pace, speed, distance, time)
- **Live Updates** — Widget refreshes automatically when you save a new preset

## Tip Jar
- **In-App Tips** — Three consumable tip options ($0.99, $2.99, $4.99) to support development

## Design
- **Dark Mode Only** — App uses a fixed dark color scheme
- **iPhone Only** — Optimized for iPhone in portrait and landscape
